### Prerequisite
Install nodejs 
- [NodeJS](https://nodejs.org/en/)

- NPM

Install dependencies

```shell
npm install
```

copy .env.sample to .env 
update the credentials in .env file 

Run the script

```
npm run start 
```
Run seeders 
```
npm run seed
```

open the browser and hit the url 
http://localhost:3000/weather
